☁️ Connect Cloud Sync

Sync your data across every device in real time using Firebase — free for small businesses.

Create a free Firebase project

Go to console.firebase.google.com, click Add project, give it a name, and finish setup.

Create a Realtime Database

In your project, go to Build → Realtime Database → Create database. Choose Start in test mode.

Get your Firebase config

Go to Project Settings (gear icon) → Your apps → Add app → Web. Register the app, then copy the firebaseConfig object and paste it below.

{ apiKey: "...", authDomain: "...", databaseURL: "...", ... }
